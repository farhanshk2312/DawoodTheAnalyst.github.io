Network Intrusion Detection (Wireshark Data)

Objective:

Detect and classify suspicious network traffic in real-time using ML models trained on Wireshark packet data.


Summary Introduction:

This Project transforms raw Wireshark packet captures into actionable cybersecurity insights. Network traffic is streamed via Kafka, processed in PySpark, and analyzed by a TensorFlow autoencoder trained to flag anomalies and potential intrusions. Deployed through a GraphQL API and visualized in Grafana, the system provides an intelligent intrusion detection pipeline capable of adapting to evolving cyber threats.

Real-Time Fraud Detection using Blockchain Data

Objective:

Build a real-time fraud detection pipeline analyzing blockchain transactions (e.g., Bitcoin or Ethereum) to identify anomalous or suspicious behavior.


Summary Introduction:

This Project analyzes blockchain transaction streams from APIs like BlockCypher and Etherscan to detect abnormal wallet behavior. Data flows through Kafka into Hadoop for scalable processing, where TensorFlow models trained on transactional patterns identify potential fraud in real-time. The results are exposed via a GraphQL API and visualized on an analytics dashboard, offering an intelligent defense layer for decentralized financial ecosystems.
