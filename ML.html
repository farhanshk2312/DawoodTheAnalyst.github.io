Network Intrusion Detection (Wireshark Data)

Objective:

Detect and classify suspicious network traffic in real-time using ML models trained on Wireshark packet data.


Summary Introduction:

This Project transforms raw Wireshark packet captures into actionable cybersecurity insights. Network traffic is streamed via Kafka, processed in PySpark, and analyzed by a TensorFlow autoencoder trained to flag anomalies and potential intrusions. Deployed through a GraphQL API and visualized in Grafana, the system provides an intelligent intrusion detection pipeline capable of adapting to evolving cyber threats.

Objective:

Predict traffic congestion levels in real-time using live data from Google Maps API combined with weather and event data.

The system uses Kafka for streaming, Hadoop for data storage, and LSTM models in TensorFlow to predict congestion up to 30 minutes ahead. Through an interactive Streamlit dashboard and GraphQL queries, users can visualize dynamic traffic heatmaps and anticipate peak hours for better route planning.

Objective:

Build an ML system that combines RSI, Volume Profile, and FII/DII activity to forecast stock movements in Indian markets.


Summary Introduction:

This project aims to build a machine learning model that predicts whether a stock will move up or down in the near term (next day or next hour) based on technical momentum indicators (RSI), volume concentration (volume profile), and institutional investor behavior (FII/DII net flows).
